# Strategic Capability Plan: "Snapshot" Lead Magnet

## Objective
To develop a standalone Chrome Extension that serves as a viral marketing tool. This extension will capture full-page web screenshots and inject a "Generated by tAImetrack" proprietary watermark.

## Business Justification (EVOLUTION.md)
This feature has been formally parked in the `EVOLUTION.md` backlog under the **Traction & Ecosystem** phase with high priority for future marketing cycles. It acts as a massive top-of-funnel conversion tool capturing free-tier users into the premium offline-first CRM funnel.

## Architectural Guidelines (DO NOT IMPLEMENT YET)
When execution is eventually cleared, the system will abide by:
1.  **Strict Isolation:** The extension source must live in a separate `/snapshot-extension` package. It will not leak or directly hook into the core `frontend/` CRM repo.
2.  **No Auth Dependency:** It operates purely client-side without any Supabase user session dependency.
3.  **Cross-Origin Compliance:** Uses native Chrome `chrome.tabs.captureVisibleTab` API or Canvas serialization.
